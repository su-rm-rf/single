# 微服务App架构
基于Single-SPA和Webpack
1个中心基座+3个子应用（1个2C前台，1个2B后台，1个2B后台分支）

# 技术栈：
基座：Vue
2C前台：React
2B后台：Vue
2B后台分支：React
